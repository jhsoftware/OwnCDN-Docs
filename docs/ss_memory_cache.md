# UI / Service Status - Memory Cache

[#INCLUDE](parts/ss_id_type.md)

## Files cached

The number and total size of the files currently cached.

Click the "Flush" button to clear out all the files.

## Cache hits

The number of requests served from the cache vs. the total number of requests, percentage, since date/time.

Click the "Reset" button to reset these counters.
