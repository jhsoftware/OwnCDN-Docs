# UI / Edit Replication Source

## Name

A descriptive name for the replication source.

## API URL

URL of the HTTP API service on the source OwnCDN server.

## API Key Value

The API Key value on the source OwnCDN server.

## Update (button)

Click to save / update the information on the page.
Changes are NOT saved until you click this button.

## Delete (button)

(only visible for existing Replication source)

Click this button to delete the Replication Source.
