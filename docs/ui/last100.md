# UI / Last 100 HTTP Request

This page show a list of the last 100 HTTP requests processed by a service.

Available for [Public HTTP Binding](../service-types/public-http-binding.md), [HTTP UI](../service-types/ui.md), and [HTTP API](../service-types/http-api.md) service types from the [Service Status page](../ui/service-status/index.md).
