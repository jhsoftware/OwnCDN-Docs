
## Requests served

A graph showing number of requests / bytes served.

Select Period / Type for various graph data.


## Relationships

Shows a map of related services - if the service is the source of another server, and/or if the services is using other services as source / storage.

The current service is indicated with a grey background.

Click on a related service to go to the Service Status page for that service.


## Edit service (button)

Click this button to edit the service.
